# Masque: Ellesui

Masque: Ellesui is a square Masque skin for World of Warcraft that closely matches the Ellesui action bar look.

## Features

- Medium-grey raised bevel frame with subtle inner shadow
- Thin black outer border baked into the frame art
- 5.5% icon zoom tuned to match the default Ellesui presentation
- Single focused skin profile designed for action bars, bags, and aura buttons

## Included Skin

- Ellesui

## Requirements

- World of Warcraft retail
- Masque

## Installation

1. Download the latest release zip.
2. Extract `Masque_Ellesui` into your `World of Warcraft\_retail_\Interface\AddOns` folder.
3. Restart the game or run `/reload`.
